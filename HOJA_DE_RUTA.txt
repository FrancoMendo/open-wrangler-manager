# OPEN WRANGLER MANAGER - Idea Principal y Hoja de Ruta

## 💡 Idea Principal
Una aplicación de escritorio construida con Tauri y React que sirve como panel de control centralizado para desarrolladores de Cloudflare Workers. La app permite gestionar múltiples proyectos (Workers) distribuidos en una estructura de carpetas local, sin necesidad de navegar manualmente por la terminal para cada despliegue o revisión de logs.

## 🚀 Funcionalidades Clave (Implementadas)
1. **Escaneo Recursivo:** Análisis profundo de directorios para encontrar archivos `.toml`.
2. **Extracción Automática:** Identificación del nombre del worker y sus entornos (`env`) directamente desde el archivo de configuración.
3. **Dashboard Visual:** Interfaz premium con tarjetas interactivas para cada worker encontrado.
4. **Terminal Integrada:** Consola xterm.js que permite ejecutar comandos de Wrangler y ver la salida en vivo.
5. **Acciones Rápidas:** Botones para `deploy` y `tail` (logs) por cada worker.
